Welcome to Python SFTP client.

Python SFTP Client enables operate SFTP remote server. manipulate files over sftp server. copy files from remote to local path. get remote file information. manage remote files. Create directory to remote machine, Remove directory to remote machine, file listings & permission:

Installation instruction using source code

$ git clone git@github.com:cis-yogesh/Python_sftp_client.git
$ python setup.py install

Installation instruction using python pip

$ pip install python_sftp_client

quick start guideline

  • import SFTP module import sftp
  • create new instance of SFTP remote server obj = sftp.Connection(host = hostname,port = port, username = username, password = password)
  • get a list of files for the given path obj.listdir(path='.')
  • change the current working directory on the remote obj.chdir(self, path)
  • get the current working directory on the remote obj.getcwd()
  • Copies a file between the remote host and the local host obj.get(remotepath, localpath = None)
  • Copies a file between the local host and the remote host. obj.put(localpath, remotepath = None)
  • get File size in bytes obj.get_file_size(remotepath)
  • Return the File created datetime obj.get_file_created_data(remotepath)
  • Open given path File and return file object obj.open(filename, mode='r', bufsize=-1)
  • Re-name old name to New name also work for move file obj.rename(oldpath, newpath)
  • Create New dir on given path, as given file permission obj.mkdir(path, mode=0777)
  • Change the permision of path obj.chmod(path, mode)
  • Remove the given path directory obj.rmdir(path)
  • Remove the given File path obj.remove_file(path)
  • Closes the connection and cleans up obj.close()
